# Platform Context

First add the dependency to your project:

```kotlin
repositories {
    mavenCentral()
}

dependencies {
    implementation("id.feinn.azisanw19:feinn-platform-context:$kmp_utils_version")
}
```

Make sure to replace $kmp_utils_version with the appropriate version of the library.
